CCREx is a module for predicting on protein sequences using CCREx models.

Currently it requires ```python=3.9```

## Installation

You can install the module using pip:

```bash
pip install ccrex
```

## Usage
```bash
ccrex -i path/to/fasta [-o /output/directory/ ] [-p/--plot True/False]
```
